# HydExplore — MapLibre React Native POC

A map-based location explorer for Hyderabad built with React Native + Expo + MapLibre.
Inspired by [mapcn](https://github.com/AnmolSaini16/mapcn), recreated for React Native.

---

## 📸 Features

- 🗺️  Full MapLibre map centered on Hyderabad
- 📍 5 clickable landmark markers (Charminar, HITEC City, Gachibowli, Madhapur, Ameerpet)
- 📷  Camera fly-to animation on marker tap
- 🧾 Animated bottom info card per location
- 🔘 Horizontal chip row for quick navigation
- 📌 Current location button + user position on map
- 🔐 expo-location permission flow

## 🚀 Setup & Run

### 1. Create a new Expo project and replace files

```bash
npx create-expo-app HydExplore -t expo-template-blank-typescript
cd HydExplore
```

Copy the provided `App.tsx`, `app.json`, `package.json`, `tsconfig.json`, `babel.config.js` into the project root.

### 2. Install dependencies

```bash
npx expo install @maplibre/maplibre-react-native expo-location
```

Or install all from scratch:

```bash
npm install
```

### 3. For iOS — install pods

```bash
npx expo prebuild --platform ios
cd ios && pod install && cd ..
```

### 4. Run on Android

```bash
npx expo run:android
```

Or with Expo Go (limited MapLibre support — native build preferred):

```bash
npx expo start
```

### 5. Run on iOS

```bash
npx expo run:ios
```

---

## 📱 Using Expo Go vs Native Build

| | Expo Go | Native Build |
|---|---|---|
| MapLibre maps | ❌ Not supported | ✅ Full support |
| expo-location | ✅ | ✅ |
| Recommendation | Dev testing only | **Use for full POC** |

> MapLibre requires native modules. Use `npx expo run:android` or `npx expo run:ios` for the full experience.

## 🗺️ Map Style

This POC uses the free MapLibre demo tile style:
```
https://demotiles.maplibre.org/style.json
```

For production, consider:
- [Maptiler](https://cloud.maptiler.com/maps/) (free tier available)
- [Stadia Maps](https://stadiamaps.com/)
- Self-hosted OpenMapTiles
